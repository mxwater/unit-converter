🌎 Unit Converter App 🇺🇸➡️🇧🇷
📌 Description
This is a simple unit converter app built with Flask that helps users convert common units like temperature, weight, volume, and distance.

I created this project because I live in the U.S. but I'm from Brazil, and I often need to convert measurements between imperial and metric systems. This app makes it quick and easy! 🚀

🔥 Features
Convert:
🌡️ Temperature (Fahrenheit to Celsius)
⚖️ Weight (Pounds to Kilograms)
🥤 Volume (Ounces to Milliliters)
🛣️ Distance (Miles to Kilometers)
Simple web interface using Flask.
Instant conversion results.
🛠️ Technologies Used
Python (Flask)
HTML & CSS (for the web interface)
